Class ListComponent

List component

For items, use a template named itemTemplate to contain content for each row. For each item in the items array, the expansion can be disabled by setting disabled to true on the item. If using actions, use a template named actionTemplate to contain expandable content for the actions of each row. If using expanding rows, use a template named itemExpandedTemplate to contain expandable content for each row.
