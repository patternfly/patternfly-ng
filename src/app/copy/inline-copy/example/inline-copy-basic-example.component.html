This example shows how the inline copy component looks when surrounded by text content. Inline Copy only occupies as much space as necessary to display its content. Font size for this component is reduced slightly relative to the surrounding text. It is recommended to not repeat the copyValue text for the tooltip as this would be redundant. Just as with BlockCopy, you have full and independent control over tooltips, aria labels, button labels, and every other piece of text that's displayed as part of this component.
